# Student Path Guide

## Description

Student Path Guide is an application designed to help high school learners make informed decisions about their future careers and university pathways.

Many students—especially those from underprivileged backgrounds—lack access to accurate and structured information about career options, university requirements, and subject choices. This often leads to poor decisions that affect their long-term opportunities.

This project aims to bridge that gap by providing students with the tools and guidance they need to confidently choose a career path that aligns with their interests and academic performance.

## feature

-🔍 Course Finder
Helps students discover courses they qualify for based on their subjects and results
-🤖 AI Chat Assistant
Provides guidance, answers questions, and explains career options in a simple way

## Usage

- Run the application
- Enter your subjects and results (if applicable)
- Explore recommended courses
- Use the AI assistant to ask questions about careers and study paths

## Why It was Built

We built this project to address the lack of accessible career guidance for high school students, particularly those from under-resourced schools. The goal is to make career information more accessible, interactive, and easy to understand.
